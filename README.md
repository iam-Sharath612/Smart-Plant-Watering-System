# 🌱 Smart Plant Watering System 💧

## 📖 Overview
The Smart Plant Watering System is an Arduino-based project that automatically monitors soil moisture and waters plants whenever the soil becomes dry. This helps conserve water, reduces manual effort, and promotes healthy plant growth.

## ✨ Features
- 🌿 Automatic soil moisture monitoring
- 💧 Automatic water pump control
- Reduces water wastage
- ⚡ Low-cost and energy-efficient
- 🤖 Minimal human intervention

## 🛠️ Components Used
- 🔹 Arduino Uno
- 🔹 Soil Moisture Sensor
- 🔹 Water Pump
- 🔹 Relay Module
- 🔹 Jumper Wires
- 🔹 Power Supply

## ⚙️ Working
1. 🌱 The soil moisture sensor continuously measures the moisture level.
2. 📊 Arduino compares the moisture value with a predefined threshold.
3. 🚰 If the soil is dry, the relay switches ON the water pump.
4. Water is supplied to the plant.
5. ✅ Once the required moisture level is reached, the pump automatically turns OFF.
